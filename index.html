MuleSoft Developer Learning Path
Designing and Consuming APIs

Describe the lifecycle of the “modern API”. Use RAML to define API resources, nested resources, and methods.

Building API Implementation Interfaces

Manually create a RESTful interface for a Mule application. Generate a REST Connector from a RAML specification.

Transforming Data With Dataweave

Write DataWeave scripts to convert JSON, XML, and Java data structures to different data structures and data types. Use DataWeave functions.

Deploying and managing APIs and Integrations

Package Mule applications for deployment. Deploy applications to CloudHub. Use CloudHub properties to ensure deployment success.
